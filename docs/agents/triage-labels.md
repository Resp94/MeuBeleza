# Rótulos de triagem

As skills trabalham com cinco papéis canônicos de triagem. Este arquivo relaciona esses papéis aos nomes reais dos rótulos usados no rastreador de issues deste repositório.

| Rótulo em `mattpocock/skills` | Rótulo em nosso rastreador | Significado                                              |
| ----------------------------- | -------------------------- | -------------------------------------------------------- |
| `needs-triage`                | `needs-triage`             | A pessoa responsável precisa avaliar a issue             |
| `needs-info`                  | `needs-info`               | Aguardando mais informações de quem relatou a issue      |
| `ready-for-agent`             | `ready-for-agent`          | Totalmente especificada e pronta para um agente autônomo |
| `ready-for-human`             | `ready-for-human`          | Exige implementação humana                               |
| `wontfix`                     | `wontfix`                  | Não será atendida                                        |

Quando uma skill mencionar um papel — por exemplo, “aplique o rótulo de triagem para trabalho autônomo” — use o rótulo correspondente nesta tabela.

Edite a coluna da direita caso seja necessário adequá-la ao vocabulário utilizado no rastreador.
